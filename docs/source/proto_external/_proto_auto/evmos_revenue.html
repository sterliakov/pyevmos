Query

Query defines the gRPC querier service.

Method NameRequest TypeResponse TypeDescription
Revenues QueryRevenuesRequest QueryRevenuesResponse

Revenues retrieves all registered revenues
Revenue QueryRevenueRequest QueryRevenueResponse

Revenue retrieves a registered revenue for a given contract address
Params QueryParamsRequest QueryParamsResponse

Params retrieves the revenue module params
DeployerRevenues QueryDeployerRevenuesRequest QueryDeployerRevenuesResponse

DeployerRevenues retrieves all revenues that a given deployer has registered
WithdrawerRevenues QueryWithdrawerRevenuesRequest QueryWithdrawerRevenuesResponse

WithdrawerRevenues retrieves all revenues with a given withdrawer address
